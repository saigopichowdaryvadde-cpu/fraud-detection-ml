# 💳 Fraud Detection using Machine Learning

## 📌 Overview

This project focuses on detecting fraudulent transactions using machine learning techniques. The goal is to identify suspicious patterns in transaction data and classify them as fraudulent or legitimate.

## 🛠️ Tools & Technologies

* Python
* Pandas
* NumPy
* Scikit-learn
* Matplotlib / Seaborn

## 📊 Features

* Data cleaning and preprocessing
* Exploratory Data Analysis (EDA)
* Model training using multiple ML algorithms
* Performance evaluation (Accuracy ~90%)

## 📈 Results

* Achieved approximately **90% accuracy** in detecting fraudulent transactions
* Compared multiple models and selected the best-performing one

## 📂 Project Structure

* `fraud_detection.ipynb` → Main notebook
* `dataset.csv` → Dataset (sample or full)
* `README.md` → Project documentation

## 🚀 How to Run

1. Clone the repository
2. Install required libraries
3. Run the Jupyter Notebook
